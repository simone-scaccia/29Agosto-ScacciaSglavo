research_ddg_tool

DuckDuckGo-based web search tool.

Exposes a CrewAI tool function to perform text searches on DuckDuckGo and return the top results in a simple formatted string.

search_rag_flow.tools.research_ddg_tool.research_ddg_tool.func(query)

Run a DuckDuckGo search and return the top three results.

Return type:

str

Parameters:

query (str)

Args:

query: The text query to search for on DuckDuckGo.

Returns:

A string with the first three results, each including title, URL, and snippet, separated by blank lines.

Note

This function is available as a CrewAI tool via the @tool decorator
